
Ccsv

A pure-C CSV parser.

== Installation

  gem install ccsv

== Usage

  require 'rubygems'
  require 'ccsv'

  Ccsv.foreach("data.csv") do |line|
    # Do something with the line array
  end

  # Or tab-separated rather than comma-separated:
  Ccsv.foreach_tab_separated("data.tab") do |line|
    # Do something with the line array
  end
